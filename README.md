## 🇬🇧 English

### 📌 About the Project
**UK Salary Calculator 2026** is a modern, lightweight, and responsive web application designed to help users in the United Kingdom calculate their net (take-home) pay. By inputting the hourly wage and weekly working hours, the app instantly provides a detailed breakdown of gross pay, taxes, and other deductions across daily, weekly, monthly, and annual timeframes.

### ✨ Features
* **Real-time Calculations:** Instant UI updates as you type, with no page reloads required.
* **Up-to-date Tax Logic (2026 Ready):** Calculates standard UK Income Tax (20%, 40%, 45% bands) and National Insurance (Class 1 at 8% and 2%).
* **Optional Deductions:** * Auto-enrolment workplace pension toggle (standard 5% calculation).
  * Student Loan Plan 2 repayment toggle.
* **Modern UI/UX:** Clean, gradient-based design with smooth hover effects and responsive layout for mobile devices.

### 🛠️ Built With
* **HTML5:** Semantic structure.
* **Vanilla CSS3:** Custom styling, Flexbox, and CSS transitions (No external frameworks).
* **Vanilla JavaScript (ES6+):** Core business logic and DOM manipulation.

### 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/wbtist/UK-Salary-Calculator-2026.git`
2. Open the folder and double-click `index.html` to run it in your default web browser.

## 🇭🇺 Magyar

### 📌 A projektről

Az órabér és a heti munkaóra megadásával az alkalmazás azonnal, részletesen lebontja a bruttó bért, az adókat és az egyéb levonásokat napi, heti, havi és éves szinten.
